## Method 1: Browser Print to PDF (Recommended - Easiest)

### Steps:
1. **Open the HTML file** in your web browser:
   - Double-click `Bershaw_Recruitment_Complete_Workflow.html`
   - Or right-click → Open with → Your browser (Chrome, Edge, Firefox)

2. **Print to PDF**:
   - Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
   - Or go to Menu → Print

3. **Select PDF destination**:
   - **Chrome/Edge**: Select "Save as PDF" from the destination dropdown
   - **Firefox**: Select "Microsoft Print to PDF" or "Save to PDF"
   - **Safari**: Select "Save as PDF"

4. **Configure settings** (optional):
   - Set margins to "Default" or "Minimum"
   - Enable "Background graphics" to include colors
   - Set paper size to A4

5. **Click Save** and choose your location
